# eslint-config-typescript

## Installation

1. Run the install command
   ```bash
   $ npm install -D @pres-inc/eslint-config-typescript
   ```
2. You also need to install peer dependencies
   ```bash
   $ npm install -D eslint
   ```

## Usage

`.eslintrc.yaml`

```yaml
extends:
  - "@pres-inc/eslint-config-typescript"
```
